# Regra importante para as escolhas

As opções não devem ser divididas apenas entre uma resposta obviamente boa e outra obviamente ruim.

Uma boa escolha deve apresentar vantagens e perdas.

Salvar um chefe pode render um aliado, mas oferece menos experiência.

Matar o chefe pode fornecer uma arma rara, mas fecha missões.

Contar a verdade pode destruir uma amizade no momento, mas fortalecê-la posteriormente.

Mentir pode resolver o problema imediatamente, mas criar uma ameaça no futuro.

Aceitar ajuda pode salvar o protagonista, mas criar uma dívida.

Recusar ajuda pode aumentar o respeito de Celão, mas tornar a missão mais difícil.

Assim, o jogador precisa escolher de acordo com o tipo de Dinis que deseja construir, e não apenas procurar a opção claramente correta.
